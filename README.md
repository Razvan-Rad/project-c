<h1>Container</h1>
<h4>Update config</h4>

```
docker-compose build devenv devweb
```
<h4>Run container</h4>

```
docker-compose up devenv
OR
docker-compose up devweb
```

<h4>attach</h4>

Windows
```
docker attach devenv
```
Debian

```
sudo docker exec -t -i devweb /bin/bash
```
